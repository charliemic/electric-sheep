# GitHub Issue Template: License Change Evaluation

**Use this template when considering a license change for commercialization**

## Issue Title

```
[LEGAL] Evaluate License Change for Commercialization
```

## Issue Body

```markdown
## Purpose

Evaluate whether to change license from MIT to proprietary license for commercialization.

**Current License**: MIT License  
**Proposed License**: [Proprietary / Dual License / Other]  
**Reason**: [Commercialization / Code Protection / Other]

## Important Note

⚠️ **MIT License already allows commercial use** - License change may not be necessary.

- [ ] Understand implications
- [ ] Estimate legal costs

- [ ] Determine if permission needed
- [ ] Contact contributors if needed

- [x] Identify any problematic dependencies

- [ ] Explain rationale
- [ ] Address community concerns
- [ ] Set expectations
- [ ] Plan timeline
```
